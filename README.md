### 🚀 Featured Project

<table>

<td width="100%">

**🛰️ AI-Based Space Anomaly Detection System**

An intelligent system that analyzes massive streams of satellite and space-sensor data to automatically flag unusual or unexpected events in space — in real time, without humans manually scanning feeds.

**Detects:**
- ⚡ Sudden radiation spikes
- 🛸 Unknown moving objects
- 💥 Space debris collision risks
- 📡 Unusual cosmic signals

**Approach:** Machine Learning + Deep Learning models trained to learn *normal* patterns of orbital paths, radiation levels, and signal behavior — then flag deviations the moment they happen.

> Instead of humans manually scanning data, the AI continuously monitors space and alerts scientists to potential threats or discoveries — instantly.
